# Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/linux-system-toolkit.git
```

Go into project directory:

```bash
cd linux-system-toolkit
```

Make scripts executable:

```bash
chmod +x toolkit.sh
chmod +x scripts/*.sh
```

Run the toolkit:

```bash
./toolkit.sh
```
